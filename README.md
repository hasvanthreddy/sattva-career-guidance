# 🚀 AI Career Navigator

AI Career Navigator is an intelligent career guidance platform that helps students explore, understand, and choose the right career path using AI-powered insights and personalized recommendations.

---

## 🌟 Overview

Choosing a career is one of the most important decisions for students. This platform simplifies that process by combining:

* 🤖 AI-based guidance
* 🎯 Personalized recommendations
* 👨‍🏫 Human guidance support

It is designed especially for students at **intermediate and above levels**.

---

## ✨ Features

### 🧠 AI Career Guidance

* Smart career suggestions based on user interests
* Personalized recommendations for each user
* Clean, structured AI responses

### 🤖 AI Mentor – *Ask Sattva*

* Interactive chatbot for career doubts
* Provides guidance on skills, careers, and opportunities

### 👨‍🏫 Real Guider

* Option to connect with experienced mentors
* Human support along with AI

### 📄 Resume Analyzer

* Upload PDF/DOCX resumes
* Extracts and displays important details

### 🔐 User Authentication

* Google Login integration
* Secure user-specific data handling

### 🎓 Student-Focused Guidance

* Intermediate-level guidance:

  * JEE / EAMCET preparation
  * MPC cutoff insights
* Personalized suggestions based on standard and interests

### 🔔 Smart Experience

* Resume where you left off
* Notifications for user progress
* Indian Standard Time (IST) support

### 🌐 Multi-Language Support

* Dynamic language handling for better accessibility

---

## ⚙️ How It Works

1. User logs in using Google
2. Selects standard (Intermediate or above)
3. Chooses area of interest
4. Gets personalized career suggestions
5. Can interact with:

   * AI Mentor 🤖
   * Real Guider 👨‍🏫
   * Resume Analyzer 📄

## 📦 Setup Instructions

```bash
git clone https://github.com/your-username/ai-career-navigator.git
cd ai-career-navigator
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file and add:

```
OPENAI_API_KEY=your_api_key
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
GOOGLE_CLOUD_BUCKET=your_bucket_name
```
